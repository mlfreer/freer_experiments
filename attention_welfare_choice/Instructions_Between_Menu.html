In this block you will be presented with a choice between menus of options. There are two equally likely states of the world: Heads and Tails. The actual state will be determined by a virtual coin flip (using the computer’s random number generator). In each decision problem you will be presented with two menus to choose from.

Fig: Interface


For each menu you see: the choice you made from this menu in the first block (top part), and the actual composition of the menu (bottom part). That is, in the example above if we consider the menu on the right, there are three options present:
  • £X payment if Heads and £Y payment if Tails,
  • £X payment if Heads and £Y payment if Tails,
  • £X payment if Heads and £Y payment if Tails.
The alternative chosen was:
  • £X payment if Heads and £Y payment if Tails,
When you choose the menu, you do not have an option to revise the choice from the Block I. That is, if you choose the menu on the left, then the corresponding chosen option would be
  • £X payment if Heads and £Y payment if Tails,
